# Debt Payoff Calculator

A C++ console application that generates a personalized debt payoff plan with the Debt Snowball or Debt Avalanche method.

## Features:
-Gathers and validates user input for multiple debts
-Debt Snowball or Debt Avalanche payoff methds
-Calculates the time to pay off each individual debt
-Generates a formatted table of the results in a text file
-Prints the table to the console

## Installation

Requires C++17 or later

Clone and build:

```bash
git clone https://github.com/ldmetz/Debt-Calculator.git
cd Debt-Calculator
g++ -std=c++17 Source.cpp -o Debt-Calculator
```
